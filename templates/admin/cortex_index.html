Funcionalidades de Gestión Social

Esta herramienta es una suite completa: la base es una base de datos donde se almacena la información. Los analistas cargan y gestionan los datos en la herramienta; como opción alternativa existe la importación desde CSV o Excel. La información se normaliza y el flujo abarca desde la carga hasta la analítica y el dashboard, con el objetivo de evitar el uso de Excel como base del proceso.

Funciones disponibles:

  • Solicitudes GRS — Listado y gestión de todos los casos. Ver, filtrar y buscar por número de caso, beneficiario o cédula; cargar datos en la herramienta (opción alternativa: importar desde CSV/Excel); exportar; y usar la acción Recalcular Scoring Manualmente para actualizar prioridad (CRÍTICO / MEDIO / BAJO).
  • Normalización y scoring — Al cargar o importar, los datos se normalizan y se aplica el scoring de prioridad de forma automática.
  • Filtros y búsqueda — Filtrar por nivel de prioridad, mes y estatus; buscar por número de caso, cédula o beneficiario.
  • Dashboard — Gráficos y KPIs (ejecución, casos totales/críticos, Pareto por especialidad, estatus, top proveedores) en Dashboard.

Debajo encontrará el acceso a Gestión Social. Haga clic en Solicitudes GRS para comenzar.
